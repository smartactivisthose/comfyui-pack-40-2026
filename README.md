## ⚡ Быстрый старт

1. **Двойной клик** по **`comfyui-pack-40-2026.exe`** в корне (или **`START.bat`**)
2. **[Releases → Latest](./releases/latest)** — тот же установщик
3. Дождитесь загрузки моделей при первом запуске
